# OUMI — صفحات GitHub Pages

هذه الملفات جاهزة للنشر على GitHub Pages. لإطلاق الموقع:

1. الصق الملفات التالية في جذر المستودع (branch main):
   - `index.html`
   - `404.html`
   - مجلد `css/` مع `style.css`
   - ملف `.nojekyll` (فارغ) لمنع معالجة Jekyll إذا لزم
2. ادفع التغييرات إلى الفرع `main`.
3. في GitHub: افتح المستودع → Settings → Pages وتأكد أن:
   - Source: Branch = `main`
   - Folder: `/ (root)` أو `/docs` حسب مكان وضع الملفات
4. انتظر دقيقة أو دقيقتين ثم افتح: `https://YASSIR1778.github.io/OUMI/`

إذا أردت أن أجهز ملفات أخرى (مثل صفحات فرعية، صور، أو إعداد CNAME) أخبرني بما تريده وسأكتبها لك جاهزة أيضاً.
